## ğŸš€ How to Run This Project

### Step 1: Train the Model

1. Open `boat-classification.ipynb` in VS Code
2. Run all cells sequentially from top to bottom
3. The notebook will save `boat_classifier_mobilenet.h5`
4. Move the model file:
   ```powershell
   Move-Item -Path "boat_classifier_mobilenet.h5" -Destination "backend/boat_classifier_mobilenet.h5"
   ```

### Step 2: Set Up the Backend

1. Navigate to backend folder:
   ```powershell
   cd backend
   ```

2. Install dependencies:
   ```powershell
   pip install -r requirements.txt
   ```

3. Start the server:
   ```powershell
   python app.py
   ```

### Step 3: Open the Frontend

Double-click `frontend/index.html` in File Explorer or open it in your browser.

### Step 4: Test the Application

1. Click "Choose Boat Image"
2. Select a boat image
3. View the prediction results!
